# SURF SCHOOL

Современный веб-сайт школы серфинга, построенный на React TypeScript с использованием архитектуры Feature-Sliced Design (FSD).

## Технологии

- **React 18** - библиотека для построения пользовательских интерфейсов
- **TypeScript** - типизированный JavaScript
- **SCSS** - препроцессор CSS с модульной структурой
- **Framer Motion** - библиотека для анимаций
- **Vite** - быстрый сборщик модулей нового поколения
- **Feature-Sliced Design** - архитектурная методология

## Структура проекта (FSD)

```
src/
├── app/              # Инициализация приложения
├── pages/            # Страницы приложения
├── widgets/           # Крупные составные компоненты
├── features/          # Функциональные возможности
├── entities/          # Бизнес-сущности
└── shared/            # Переиспользуемые компоненты и утилиты
    ├── ui/            # UI компоненты
    ├── styles/        # Глобальные стили
    └── assets/         # Статические ресурсы
```

## Установка

```bash
npm install
```

## Запуск проекта

### Режим разработки

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:8080`

Vite обеспечивает мгновенный запуск сервера и горячую перезагрузку модулей (HMR)

### Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут находиться в папке `dist/`

## Компоненты

### Widgets

- **Header** - Шапка сайта с навигацией
- **AboutSection** - Секция "О нас"
- **ActivitiesSection** - Секция с активностями
- **ReviewsSection** - Отзывы клиентов
- **Footer** - Подвал сайта
- **QuoteSection** - Цитата

### Features

- **Navigation** - Навигационное меню
- **HeaderInfo** - Информация в шапке
- **HeaderNews** - Новости в шапке
- **SocialLinks** - Социальные сети

### Shared

- **Button** - Переиспользуемая кнопка с анимациями
- **Styles** - Глобальные стили и переменные

## Анимации

Проект использует Framer Motion для создания плавных анимаций:

- Появление элементов при скролле
- Hover эффекты на интерактивных элементах
- Плавные переходы между состояниями
- Анимация мобильного меню
